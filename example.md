# 锦江学院体院汇报PPT完整风格总结

（含封面、背景、磨砂卡片、文字色彩、表格、装饰、整体调性全维度）

## 一、基底背景系统（统一哑光紫底）

1. **主底色**
   色值#42205C 低明度哑光深紫罗兰，纯平铺满，无镜面反光、无通透玻璃渐变，雾面磨砂质感，庄重适配院校申报公文汇报。
2. **底层弱底纹**
   低透明度（8%-12%）白色细碎小圆点，平铺在深紫底层，虚化朦胧，仅微弱丰富层次，不会遮挡文字，远看依旧干净纯色，杜绝花哨运动大图、实景照片、渐变花纹。
3. **统一规则**
   全程不换背景，整套PPT所有内页、封面共用同一款哑光深紫基底，视觉连贯统一。

## 二、封面专属版式设计

1. **排版结构（居中极简对称）**

- 页面视觉中心：超大加粗白色无衬线主标题（如《体育学院专业设置调整方案》），是第一视觉重心；
- 主标题下方：浅灰小字副标题「xxx」，层级弱化；
- 中间分割：一条细浅紫色水平实线，隔开标题区与页脚；
- 页脚左右分栏：左侧xxx全称（浅灰小字）、右侧汇报年月，左右平衡留白充足。

2. **色彩搭配**
   纯白主标题+浅灰辅助文字+浅紫分割线，无多余亮色点缀，极致正式、简约大气。
3. **装饰限制**
   无校徽大图、无运动人物、无立体特效，纯文字+细线条构成封面，政务汇报克制感拉满。

## 三、核心视觉：磨砂信息卡片（全页面承载内容的标准组件）

### 1. 卡片底色与质感

色值#532B73，比背景浅一阶同色系雾面紫，**不透光哑光磨砂块**；关闭渐变、发光、阴影、浮雕、3D、玻璃高光，纯扁平纯色填充，可轻微添加细杂点纹理强化磨砂雾感。

### 2. 卡片形态与布局

- 形状：标准直角矩形，拒绝圆角、异形；
- 排布：双栏/三等分均匀并列，块与块预留大量留白，不贴页面边缘；
- 功能分区：分为单专业独立小卡片、页面底部通栏总结大卡片两种尺寸。

### 3. 卡片分割装饰（仅两种线条）

1. 侧边竖装饰条：高亮金黄#F9D240窄竖线，放在卡片最左侧，用作板块标识，窄条小面积不抢视线；
2. 内部分割线：浅紫色#9D78C2极细水平实线，只做行间分层，全部删除竖边框，保持清爽。

## 四、完整文字色彩层级（三色体系，规避红黄冲突）

整套PPT只使用4类文字色，分工清晰，严格区分主次：

1. **纯白 #FFFFFF（一级视觉）**
   页面大章节标题（3.1/3.4）、卡片内模块标题、表格表头，加粗无衬线黑体，对比度最高；
2. **金黄 #F9D240（二级重点高亮）**
   生源缺口数字、各级在校生人数、核心数据，仅用于数字与关键论据，小面积点缀；
3. **浅灰紫 #D1C4E0（三级辅助说明）**
   正文解释文字、备注（转入/退学/暂停招生）、数据来源，弱化存在感；
4. **暗酒红 #9E2B42（仅总计专属）**
   全院总人数、核心规划标题（适度扩大招生规模），只用于汇总、核心小标题，不与黄色大面积同屏，解决红紫黄冲突问题。

## 五、表格专属规范（适配磨砂卡片的数据展示标准）

1. **载体**：整张表格内嵌在#532B73磨砂卡片内部，不单独使用白色表格块割裂背景；
2. **表头设计**：表头填充加深紫#4C2668，白色加粗表头文字，区分数据行；
3. **边框规则**：仅保留水平细浅紫分割线，全部删除竖线，线条粗细0.75磅；
4. **单元格**：加大上下左右内边距，文字宽松不拥挤；
5. **文字配色**：xxx名白色，特别数字金黄，备注浅灰紫，底部总计行暗酒红加粗突出。

## 六、装饰图标与点缀元素（极致克制）

1. 仅使用单色极简线性扁平图标（灯泡、人物、书本），无填充渐变、无立体效果；
2. 图标只有两种颜色：纯白/金黄，放置在底部通栏总结框左侧；
3. 全程杜绝：运动剪影、篮球跑道、人物照片、彩带、渐变光斑、立体边框等花哨体育风装饰，优先保障方案严谨政务属性。

## 七、整体风格核心调性总结

1. **简约克制**：零冗余特效、全扁平哑光设计，依靠色块+线条分层，无多余美化；
2. **专业政务学术风**：深紫基底+规整模块化卡片，适配高校专业调整、向上申报汇报场景，稳重正式；
3. **轻量体院适配**：紫色基底自带年轻活力，少量金黄提亮抵消深紫沉闷，兼顾体育专业活力，不喧宾夺主；
4. **信息优先**：所有色彩、卡片、线条设计均服务政策、数据、文字内容，视觉元素只用来分层、高亮重点，不脱离汇报主题做纯装饰。
