# 问题与反思

<Card title="当前不足" accent="#9E2B42" padding="5">
- <span class="text-desc">高水平师资引进力度有待加强，博士占比仅 <span class="text-data">18%</span></span>
- <span class="text-desc">科研平台支撑能力不足，省级以上重点实验室尚属空白</span>
- <span class="text-desc">校企合作深度不够，产教融合项目落地周期较长</span>
- <span class="text-desc">国际化办学水平有待提升，暂无联合培养项目</span>
</Card>

<Card title="改进方向" accent="#F9D240" padding="5">
- <span class="text-desc">实施"<span class="text-data">英才引进计划</span>"，力争新增博士 3-5 名</span>
- <span class="text-desc">申报省级运动健康重点实验室，搭建科研平台</span>
- <span class="text-desc">深化与 5 家体育龙头企业战略合作</span>
- <span class="text-desc">启动与国外高校的<span class="text-data">2+2</span>联合培养项目</span>
</Card>
